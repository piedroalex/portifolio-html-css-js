# Projeto - Meu Portifólio
Projeto desenvolvido durante a aula de Programação Web do curso de Tecnologia em Sistemas para Internet EAD com as tecnologias HTML5, CSS3 e JavaScript.

## Atividade Prática

### HTML5
* Cabeçalho com Menu
* Seção de Apresentação
* Seção de Projetos
* Seção de Contato
* Rodapé

### CSS3
* Criar o arquivo style.css e vincular ao HTML
* Resetar as margens padrão
* Usar Flexbox para criar um menu horizontal no header
* Centralizar o conteúdo da seção Sobre Mim e estilizar a imagem
* Transformar a seção Meus Projetos em uma grade de cards (lado a lado no desktop, um abaixo do outro no mobile)
* Estilizar o formulário de contato
* Estilizar o rodapé

### JAVASCRIPT
* Implementar um alerta ao enviar o formulário
* Criar o botão de alternância de tema no topo da página
* Renderizar dinamicamente os projetos via dados
